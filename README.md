# Invoice Merger - 发票合并工具

> 将多个发票 PDF 两两拼合为 A4 上下结构，中间加裁剪线，打印后沿虚线裁开即可。

---

## 📖 为什么做这个工具

打印发票时，一张 A4 纸打一张发票太浪费，也不符合财务归档要求（发票大小约 A4 一半）。市面上的解决方案：

- **WPS**：会员功能，要收费
- **在线工具**：要么收费，要么有水印，还要上传文件到第三方服务器

于是想：能不能用 OpenClaw 自己做一个？

说干就干，合并出来打印效果不错，又参考付费软件加上了裁剪线，方便打印后裁开。做完了想着既然好用，不如封装成 Skill 分享出来，让更多人免费用上这个功能。

**省纸、省钱、省心。**

---

## ✨ 功能特点

- **上下结构**：两个发票上下拼在一页 A4 纸上
- **自动裁剪线**：中间虚线分隔，打印后沿虚线裁开
- **奇数补齐**：奇数个文件时，最后一个自动复制补齐
- **自动命名**：输出到桌面 `YYYYMMDD-发票合并/` 目录，文件名自动加序号
- **跨平台**：支持 macOS 和 Windows

---

## 📋 处理逻辑

| 输入文件数 | 处理方式 | 输出页数 |
|-----------|---------|---------|
| 1 个 | 自我复制，上下拼 1 页 | 1 页 |
| 2 个 | A+B 上下拼 1 页 | 1 页 |
| 3 个 | A+B 拼 1 页，C+C 拼 1 页 | 2 页 |
| 4 个 | A+B 拼 1 页，C+D 拼 1 页 | 2 页 |
| 5 个 | A+B、C+D 各拼 1 页，E+E 拼 1 页 | 3 页 |

## ⚙️ 技术细节

- **排版**：A4 纸（595.27 × 841.89 pt），上下各一半
- **裁剪线**：虚线样式（6pt 实线 + 4pt 空白），灰色
- **间隙**：发票与裁剪线之间留 15pt 间隙，方便裁剪
- **缩放**：自动缩放适应 A4 宽度，保持比例

## 💡 常见问题

**Q: 只有一张发票怎么办？**

A: 会自动复制成两张拼在一页，打印后裁开就是两张相同的发票。

**Q: 支持什么格式的文件？**

A: 目前只支持 PDF 格式，每个 PDF 只取第一页。

**Q: Windows 上输出路径在哪？**

A: 输出到桌面 `YYYYMMDD-发票合并\` 目录，和 macOS 一致。

**Q: 裁剪线打印出来看不清？**

A: 裁剪线是浅灰色虚线，打印后沿虚线用剪刀或裁纸刀裁开即可。
